Instructions

Try to find the optimal path from the start node to the end node. You can generate a maze or create your own maze by clicking on the grid. The lava blocks are unwalkable and you can't pass through them. Click on the "Start Search" button to find the optimal path with A*.

Left click: Draw start and end node.
Right click: Draw lava.
